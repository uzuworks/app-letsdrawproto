# 🎨 おえかきアプリ

古の子供向けペイントソフトの機能「かいてみよう！」に
影響されて作ったWebアプリです
自由に描いたり、絵文字スタンプを貼ったり、
テーマに合わせて絵を描いたりして遊べます。

なるべくシンプルな操作感を心がけています。
